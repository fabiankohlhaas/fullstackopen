Getting started with Full Stack Open

This Website was created to refresh the basics of web development as part of the Full Stack Open course from the University of Helsinki. I followed the tutorial from mdn.

Full Stack Open teaches the fundamentals of:

  • Redux
  • React
  • Node.js
  • MongoDB
  • GraphQL
  • TypeScript
  • Express
  • Docker
